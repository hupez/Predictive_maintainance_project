
# Проект: Бинарная классификация для предиктивного обслуживания оборудования

## Описание проекта

Цель проекта — разработать модель машинного обучения, которая предсказывает, произойдёт ли отказ оборудования (Target = 1) или нет (Target = 0). Результаты работы оформлены в виде Streamlit-приложения.

Проект представляет собой веб-приложение для предиктивного обслуживания промышленного оборудования. Оно анализирует текущие параметры работы станка (температура, скорость вращения, крутящий момент, износ инструмента) и прогнозирует вероятность отказа.

**Возможности приложения:**
- Загрузка и предобработка данных (удаление лишних столбцов, кодирование категорий, масштабирование)
- Обучение 4 моделей: логистическая регрессия, Random Forest, XGBoost, SVM
- Оценка моделей по метрикам Accuracy, ROC-AUC, матрица ошибок
- Интерактивное предсказание отказа для нового оборудования
- Встроенная презентация проекта

**Практическая ценность:**  
Приложение может быть интегрировано в АСУ ТП завода. Оно позволяет:
- заранее планировать ремонт, избегая внезапных остановок производства;
- снижать затраты на обслуживание за счёт замены деталей только при реальной угрозе отказа;
- повышать безопасность работы персонала.

## Используемые технологии

- **Streamlit** — веб-интерфейс
- **Scikit-learn** — модели и метрики
- **XGBoost** — градиентный бустинг
- **Pandas / NumPy** — обработка данных
- **Matplotlib / Seaborn** — визуализация

## Результаты модели

| Модель | Accuracy | ROC-AUC |
|--------|----------|---------|
| Логистическая регрессия | 0.8210 | 0.9063 |
| Random Forest | 0.9805 | 0.9686 |
| **XGBoost** | **0.9885** | **0.9746** |
| SVM | 0.8300 | 0.9047 |

**Лучшая модель:** XGBoost (Accuracy = 98.9%, ROC-AUC = 0.97)

## Установка и запуск
1. Клонируйте репозиторий:
 git clone https://github.com/hupez/Predictive_maintainance_project.git
2. Установите зависимости:
 pip install -r requirements.txt
3. Запустите приложение:
 streamlit run app.py

## Как использовать приложение
После запуска откроется браузер с главной страницей.

1.Перейдите на вкладку «Анализ и модель».

2.Данные загрузятся автоматически

3.Выберите модель в боковом меню (логистическая регрессия, Random Forest, XGBoost или SVM).

4.Модель обучится автоматически, вы увидите метрики, матрицу ошибок и ROC-кривую.

Для предсказания нового оборудования:

1.Прокрутите страницу вниз до формы «Предсказание отказа для нового оборудования».

2.Заполните параметры (температура, скорость, крутящий момент, износ, тип продукта).

3.Нажмите «Предсказать» – получите результат (отказ / норма) и вероятность отказа.

4.Перейдите на вкладку «Презентация проекта» для просмотра слайдов с описанием проекта.
